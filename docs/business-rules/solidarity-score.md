# Score de solidarité

## Principe général

Le score de solidarité représente le taux de participation financière d’un membre aux différents évènements et cotisations du Contingent.

Ce score influence les aides futures dont le membre pourra bénéficier.

---

## Types de score

### 1. Score projectionnel

Basé uniquement sur :
- les évènements internes à la projection ;
- les cotisations internes.

Visible uniquement dans la projection concernée.

---

### 2. Score global

Basé sur :
- les évènements globaux ;
- les évènements projectionnels ;
- les cotisations générales.

Visible par tous les membres du Contingent.

---

## Principe de calcul

Exemple :

Si les montants nominaux sont :

- Officiers : 20 000
- Sous-officiers : 10 000
- MDR : 5 000

Et que le score de solidarité du bénéficiaire est de 30% :

Les montants réellement demandés deviennent :

- Officiers : 6 000
- Sous-officiers : 3 000
- MDR : 1 500

---

## Remarques

Le système devra permettre :
- les paiements partiels ;
- les statistiques ;
- les évènements généraux ;
- les évènements particuliers ;
- les cotisations périodiques.
